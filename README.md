# Calorie Burn Predictor

A machine learning model that predicts calories burned during a workout based on personal stats like age, weight, heart rate, and duration.

## What it does
- Takes workout inputs (age, height, weight, duration, heart rate, body temp)
- Predicts calories burned using a Random Forest model
- Interactive UI built with Gradio

## Model Performance
- Algorithm: Random Forest Regressor
- Mean Absolute Error: 2.51 calories

## Libraries Used
- pandas, numpy
- scikit-learn
- matplotlib
- gradio

## How to run
1. Open the `.ipynb` file in Google Colab
2. Upload `calories.csv` and `exercise.csv` when prompted
3. Run all cells
4. Use the Gradio link to interact with the app

## Dataset
Kaggle - Calories Burned Dataset : (https://www.kaggle.com/datasets/fmendes/fmendesdat263xdemos)
